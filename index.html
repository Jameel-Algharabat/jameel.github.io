Online Contracts Management System using REACT JS

×
The project is a contract management software. The landing page includes:
  • Details about system features (with reference to other sections)
  • Staff directory
  • Related YouTube videos
The user can register and login to the site. The data is stored on MySQL. After logging in, the Profile section appears, displaying the logged in user's information, including:
  • Full name, department, employee number, photo, ...
The user can update his data except for the employee number. There is a second page called "Contracts" which contains the contract grid(s) showing:
  • contract name, date signed, expiry date, total cost and the name of the legal officer and employee number
Each element of the node network has three buttons:
  • Contract Details: When clicked, a popup appears showing all details (mentioned) with Contract Service Items, Amount (Software/Hardware), Total Cost, Warranty Start Date, Warranty End Date)
  • Company Details: When clicked, a popup appears showing the company name, location, contact person name, and contact information.
Contracts are automatically colored yellow 10 days before expiry and red 3 days before expiration. Expired contracts do not appear on the user's page, only the manager can see it. The contract page contains the button "++++ Add New ++++" when clicked a popup appears to add a new contract. Newly added nodes do not.
